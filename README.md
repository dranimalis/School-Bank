# 🏦 School-Bank

School-Bank is a simple and practical project designed to simulate a **school-based banking system**. It helps manage student balances, transactions, and basic financial records in an organized and beginner-friendly way.

This project is suitable for **school projects, learning purposes, or small internal systems**.

---

## ✨ Features

- 👤 Student account management
- 💰 Balance tracking
- ➕ Deposit system
- ➖ Withdrawal system
- 📜 Transaction history
- 🔒 Simple and safe data handling
- 📊 Easy-to-read structure

## 🚀 Getting Started

### Prerequisites

Make sure you have:

- Python installed
- Git
- A modern web browser

### Installation

1. Clone the repository:

   ```bash
   git clone https://github.com/dranimalis/School-Bank.git
   ```

2. Navigate to the project folder:

   ```bash
   cd School-Bank
   ```

3. Install the required dependencies:

   ```bash
   pip install -r requirements.txt
   ```

4. Start the backend server:

   ```bash
   python app.py
   ```

5. Open the application in your browser.
