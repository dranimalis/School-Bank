# 🏦 School-Bank

School-Bank is a simple and practical project designed to simulate a **school-based banking system**.  
It helps manage student balances, transactions, and basic financial records in an organized and beginner-friendly way.

This project is suitable for **school projects, learning purposes, or small internal systems**.

---

## ✨ Features

- 👤 Student account management  
- 💰 Balance tracking  
- ➕ Deposit & ➖ withdrawal system  
- 📜 Transaction history  
- 🔒 Simple and safe data handling  
- 📊 Easy-to-read structure

---

## 🛠️ Technologies Used

- Language: *HTML, CSS & JavaScript*  
- Storage: *SQLite*  
- Platform: *Web*
